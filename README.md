# Penny - Financial Dashboard

A comprehensive financial dashboard for young adults (18-28) featuring:
- Interactive charts and analytics
- Budget tracking
- Savings goals
- Financial calendar
- AI-powered insights

## Getting Started

Simply open `index.html` in any modern web browser. No build step required.

## Features

- Dashboard with net worth overview
- Account management
- Transaction tracking
- Budget categories
- Savings goals with progress rings
- Financial calendar
- Reports and analytics
- Achievement system

Built with React, vanilla CSS, and SVG charts.
